# Phishing Email Analysis Case Study

## Overview

This project presents the analysis of three real-world phishing email campaigns obtained from the UC Berkeley Phishing Examples Archive.

The objective was to identify phishing indicators, analyze social engineering techniques, extract Indicators of Compromise (IOCs), and map findings to the MITRE ATT&CK framework.

## Project Objectives

* Analyze real phishing emails
* Identify suspicious sender information
* Examine social engineering techniques
* Extract Indicators of Compromise (IOCs)
* Perform risk assessment
* Map findings to MITRE ATT&CK techniques
* Create professional investigation reports

## Cases Analyzed

### Case 1 – Card Internship Scam

**Attack Type:** Job/Internship Scam

**Techniques Observed:**

* Authority
* Trust Exploitation
* Information Harvesting

**Risk Level:** High

---

### Case 2 – Fake Wikipedia Page Editorial Assistance Scam

**Attack Type:** Professional Recognition Scam

**Techniques Observed:**

* Flattery
* Trust Building
* Social Engineering

**Risk Level:** Medium

---

### Case 3 – Malware Hidden in an Invitation Email

**Attack Type:** Invitation-Based Phishing

**Techniques Observed:**

* Curiosity Lure
* Event Invitation Theme
* Spearphishing Link

**Risk Level:** High
